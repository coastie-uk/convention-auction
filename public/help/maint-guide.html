Maintenance Panel User Guide

Access

Log in with the dedicated maintenance password. A successful login unlocks all maintenance tools.

Available Tools

Database

  • Create Backup on Server: Saves a backup of the database on the server.
  • Restore Database: Upload and restore a `.zip` backup from your device.
  • Reset Database: Wipes all data. Requires confirmation.

Import/Export

  • Export CSV: Download all items as a CSV file.
  • Import CSV: Upload a CSV file to bulk add items.
  • Download Database and Photos: Downloads a zipped copy of the current database and all images.

Diagnostics

  • Photo Storage Report: Shows total image count and size.
  • Data Integrity Check: Reports missing photo files associated with items.
  • Clean Up Unused Photos: Identifies and removes orphaned photo files not linked to any item.

Security

  • Change Password: Update the admin password. Requires confirmation and input validation.

Generate Test Data

  • Generate: Add a specified number of sample items for testing purposes.

System

  • Restart Server: Triggers a backend restart.
  • View Server Logs: Displays recent backend logs.

Presentation Configuration

  • Edit Slide Config: Opens a JSON editor to update PowerPoint slide layout options. Includes validation and save button.

Navigation

  • Back to Admin Panel: Return to the main admin interface.
  • Log Out: Ends your maintenance session.
