Admin Page User Guide

Overview

The admin page allows authorized users to manage submitted items for the charity auction. You can add, edit, delete, export, and generate presentation materials for all items.

Functions

  • Login: Enter the admin password to access the admin interface. Sessions last for 8 hours.
  • View Items: All submitted items are displayed in a table with ID, description, contributor, artist, and thumbnail.
  • Add New Item: Use the "New Item" button to open a form where you can submit a new entry including photo and notes.
  • Edit Item: Click "Edit" next to any item to modify its information or upload a new photo.
  • Delete Item: From the edit view, you can permanently delete an item (including its image file).
  • Export CSV: Download a spreadsheet-compatible CSV file containing all item data.
  • Slides: Generate a PowerPoint presentation with one slide per item, showing all key information and photo if available.
  • Item Cards: Generate compact A6-sized cards for printing or display at the auction event.
  • Refresh: Reload the item list from the server.
  • Logout: End your admin session and return to the public submission page.
